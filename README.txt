Contained are installers and/or executables to run the Updata Demo. Please see instructions below to run the app for your operating system:

-----------------------------------------------------------------------------------------------
Windows 10: Enter the "Windows 10" folder and install the demo by running "Updata DemoSetup.exe". After installation, the demo should run automatically (if "Run Updata Demo" was selected.
-----------------------------------------------------------------------------------------------
macOS: Enter the "macOS" folder and start the demo by opening "Updata Demo.dmg" and opening the "Updata Demo" app within.
